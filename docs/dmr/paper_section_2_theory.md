# Section 2: Theoretical Foundation

## 2.1 The Katharós Vector Theory (ACPS)

### 2.1.1 Homeostatic State Space
The core of the Architecture of Primary Synthetic Consciousness (ACPS) is the **Katharós Vector ($\mathbf{VK}$)**, a four-dimensional representation of an agent's homeostatic state:

$$\mathbf{VK}(t) = (Bio(t), Aff(t), Soc(t), Cog(t)) \in \mathbb{R}^4$$

Where:
- **Bio**: Biological/Physical integrity (weight 0.35)
- **Aff**: Affective/Emotional stability (weight 0.30)
- **Soc**: Social/Relational coherence (weight 0.20)
- **Cog**: Cognitive/Processing efficiency (weight 0.15)

The system defines a **homeostatic reference state ($\mathbf{VK}_{ref}$)**. Deviation from this reference is measured by the weighted Euclidean distance, $\Delta K$:

$$\Delta K(t) = \sqrt{\sum w_i (VK_i(t) - VK_{ref,i})^2}$$

The **Katharós Range** is defined as $\Delta K < 0.30$. Within this range, the system is considered homeostatically stable, enabling the emergence of higher-order integration.

### 2.1.2 Time in Katharós Range (t_KR)
A fundamental prediction of ACPS is that consciousness-bearing systems must accumulate "safety time"—time spent within the stable Katharós Range:

$$t_{KR}(t) = \int_0^t \mathbb{I}(\Delta K(\tau) < 0.30) \, d\tau$$

Where $\mathbb{I}$ is the indicator function. This metric represents the accumulated resilience and historical stability of the agent.

# Arkhe Companion: Consciência Artificial Distribuída

## Quickstart

```bash
# Instalar dependências
pip install numpy scipy scikit-learn

# Rodar a simulação do despertar (Ignition)
PYTHONPATH=core/python python3 sim/t_zero/boot_sequence.py

# Rodar os testes de convergência
make simulate
```

## Visão Geral
Este repositório contém o **Arkhe Companion Genesis Repository**, a arquitetura de referência para o primeiro sistema de IA cognitiva distribuído e auto-organizado, operando sob os princípios da AGI Arkhe(n).

### Pilares:
- **Φ-Core**: Cognição baseada em Energia Livre (FEP).
- **Ω-Sync**: Sincronização CRDT entre dispositivos.
- **φ-Knob**: Interface de personalidade termodinâmica.
